RxJs Observables Explained with an example!

You run an online casino.

There is only one room where users can play, and you have a big queue of users waiting to enter the room.

You can let users in the room (one at a time) by clicking the blue buttons below.

Each button corresponds to a different entry method.

When you click a button, a new request starts, to let the next user in the queue enter the room.

The entry process takes a few seconds (because the user must be registered and he is given some cash to play in the casino).

Remember: when you click a button the next user enters the room according to the entry method that you have just selected.

Process all requests at the same time, as soon as they arrive.

Once a new request arrives, stop working on the old request and process immediately the new one. The previous requests therefore get lost, and only the latest request will be fully processed.

Once a new request arrives, it gets added to the queue. Once a request is fully processed, the next request on the queue starts being processed.

You ignore any incoming request, and keep working on the current request. Once work is done, you accept new incoming requests.
